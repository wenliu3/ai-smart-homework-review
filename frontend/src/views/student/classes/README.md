# 学生班级模块

## 概述

学生班级模块采用全新的双栏布局设计，提供直观高效的班级和作业管理体验。

## 功能特点

### 🎯 双栏布局设计

- **左侧班级列表**：紧凑的班级卡片展示，支持搜索和快速选择
- **右侧作业列表**：对应班级的作业详情，支持状态筛选

### 🔍 智能搜索与筛选

- **班级搜索**：实时搜索班级名称
- **作业筛选**：按状态筛选（全部/待完成/已完成）

### 📱 响应式设计

- 适配桌面端和移动端
- 流畅的交互动画
- 优雅的加载状态

### ⚡ 性能优化

- 计算属性自动缓存
- 按需加载作业数据
- 精简的组件结构

## 技术实现

### 文件结构

```
src/views/student/classes/
├── index.vue              # 主页面（双栏布局）
└── README.md              # 说明文档
```

### 核心技术

- **Vue 3 Composition API**：响应式数据管理
- **Element Plus**：UI 组件库
- **Tailwind CSS**：样式设计
- **TypeScript**：类型安全

### API 集成

- `getClassList()`：获取学生班级列表
- `getMyAssignments()`：获取班级作业列表
- `joinClass()`：加入班级功能

## 使用说明

### 基本操作

1. **选择班级**：点击左侧班级项即可选中并加载对应作业
2. **搜索班级**：在左上角搜索框输入班级名称进行筛选
3. **筛选作业**：使用右上角的状态筛选按钮
4. **查看作业**：点击作业卡片查看详细信息
5. **加入班级**：点击"加入班级"按钮输入邀请码

### 状态说明

- **待完成**：尚未提交的作业
- **已完成**：已提交并批改的作业
- **进行中**：已开始但未完成的作业
- **已逾期**：超过截止时间的作业
