# 页面头部样式统一化

## 概述

为了保持整个系统的界面一致性，我们创建了统一的页面头部组件，参考作业管理页面的现代化设计风格。

## 核心组件

### 1. PageHeader 组件

**位置**: `src/components/PageHeader.vue`

**功能**:

- 统一的页面标题和描述展示
- 灵活的操作按钮插槽
- 响应式设计，支持移动端适配

**使用方式**:

```vue
<page-header title="页面标题" description="页面描述">
  <template #actions>
    <el-button type="primary">主要操作</el-button>
    <el-button>次要操作</el-button>
  </template>
</page-header>
```

**设计特点**:

- 白色背景，轻微阴影
- 圆角设计(8px)
- 20px 垂直内边距，24px 水平内边距
- 标题字体大小 20px，加粗 600
- 描述字体大小 13px，浅灰色

### 2. SearchCard 组件

**位置**: `src/components/SearchCard.vue`

**功能**:

- 统一的搜索区域样式
- 支持搜索表单和控制按钮分离
- 响应式布局，移动端自动堆叠

**使用方式**:

```vue
<search-card>
  <template #search-form>
    <el-form :inline="true" :model="searchForm">
      <!-- 搜索表单内容 -->
    </el-form>
  </template>
  
  <template #controls>
    <!-- 控制按钮，如视图切换 -->
  </template>
</search-card>
```

## 设计规范

### 色彩规范

- **主背景**: `#f5f7fa` (浅灰蓝)
- **卡片背景**: `#ffffff` (纯白)
- **主标题**: `#303133` (深灰)
- **描述文字**: `#909399` (中灰)
- **边框色**: `#e4e7ed` (浅灰)

### 间距规范

- **页面边距**: 16px (移动端) / 24px (桌面端)
- **卡片内边距**: 16px (移动端) / 20px (桌面端)
- **组件间距**: 16px
- **按钮间距**: 8px

### 圆角规范

- **卡片圆角**: 8px
- **按钮圆角**: 6px
- **输入框圆角**: 6px

### 字体规范

- **主标题**: 20px / font-weight: 600
- **副标题**: 13px / color: #909399
- **表单标签**: 13px / font-weight: 500 / color: #606266
